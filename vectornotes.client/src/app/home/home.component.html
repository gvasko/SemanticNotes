Semantic Notes

This is a simple note taking application. The word 'semantic' means that the application can find similarities between notes based on their content.

The goal of this application is to see if a language detector approach can detect the topic as well.

It is also a goal to have a starting point to understand some very basic concepts in modern AI, like high-dimensional representations. Therefore the implementation is not entirely correct and not scientific, it is only for building some basic understanding.

The implementation is based on the following references:

  1. COMPUTING with HIGH-DIMENSIONAL VECTORS, by Pentti Kanerva, UC Berkeley
  2. Language Geometry using Random Indexing, by Aditya Joshi, Johan T. Halseth, and Pentti Kanerva, UC Berkeley

Usage

After logging in, you can create notes, navigate to the Notes tab. These will be the texts used for similarity calculation. Notes can be organized into collections, so you can experiment with different sets of texts.

Each note can have tags. A tag is a name-value pair, so you can decorate a text with its attributes, like what language it is written in or what topic it represents.

You can open a note in browser mode, where its 10 most similar notes are listed, and their tags are also shown, so you can see what the currently selected note (or text) is like.

The recommended usage is to create at least a collection that contains the reference texts, add their tags that describe them and use this reference collection on any other note to detect what a custom text is like most, e.g. what is its language or topic.

Using the Map tab, a visualisation is provided based on the level of similarity, so the more similar the texts, the closer they should be. Unfortunately this algorithm is just a quick common sense one which does not work well in many dimensions.
