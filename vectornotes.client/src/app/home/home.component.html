Welcome About this App

This is a simple note-taking application with a twist: it's semantic. That means the app can detect similarities between notes based on their content—almost as if it understands the meaning behind the text.

The main goal is to explore whether a language detection approach can also hint at the topic of a note.

Additionally, the project serves as a hands-on introduction to foundational concepts in modern AI, such as high-dimensional vector representations. The implementation is intentionally simplified and not scientifically rigorous—its purpose is to offer an accessible starting point for learning and experimentation.

 Application How It Works

After logging in, head over to the Notes tab to start creating your notes. These are the texts the app will use to measure similarity. Notes can be grouped into collections, which lets you experiment with different sets of texts.

Each note can also have tags, which are simple name-value pairs. Tags let you describe a note with extra context—like the language it's written in or the topic it covers.

You can open any note in browser mode, where the app will show its 10 most similar notes based on content. You'll also see their tags, helping you understand what the current note is most related to.

A typical workflow is to create a reference collection filled with known texts and carefully tag them. Then, when you add a new note, you can compare it against the reference set to get a sense of its language or topic—based purely on content similarity.

In the Map tab, you'll find a simple visualization that places notes closer together when they’re more similar. Keep in mind that this is just a basic algorithm for intuition—it’s not designed for high-dimensional accuracy, but may give a helpful visual overview.

Future Development When Pigs Fly?

Possible Further Improvements

Smarter Visualization: High-dimensional data is tricky to display in two dimensions. A better approach—like the one described in L. van der Maaten’s “Visualizing High-Dimensional Data using t-SNE”—could help make groupings and relationships between notes easier to see.

Adjustable Alphabet Size: Right now, the underlying "alphabet" used to generate vectors has 5,000 dimensions. It would be interesting to test how the app behaves with smaller or larger dimensional spaces.

Progress Indicators: Creating language vectors can take some time, especially for multiple notes. A simple loading or progress indicator would help users understand what’s happening in the background.

Starter Content for New Users: To explore the app’s features, users need to create a reference collection, which takes some effort. Pre-filled example notes and collections could make onboarding smoother.

Expanded Login Options: At the moment, you can log in using a Microsoft or Google account. Adding more login providers or support for anonymous usage might improve accessibility.
